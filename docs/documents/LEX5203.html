FOI Request LEX5203

Date: 2023

This FOI request sought access to daily "Postal Vote CSV file for upload" documents.

Documents from the FOI Request:
The documents released consist of a log or index of specific files, each titled "Postal Vote CSV file for upload," spanning dates from September 22, 2023, to October 11, 2023.

Main Content Related to the FOI Request:
All listed documents, which are the "Postal Vote CSV file for upload" for each specified date, were "Released in full." No exemptions were applied to any of the requested files.
